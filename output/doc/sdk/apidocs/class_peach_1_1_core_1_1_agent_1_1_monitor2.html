◆ InternalEvent

EventHandler Peach.Core.Agent.Monitor2.InternalEvent

An event handler that can be used by monitor implementations to alert others about interesting events occuring. The peach core does not make use of this event.

This event handler is completly ignroed by the peach core. It can be useful for writing tests against monitors so the testing framework can get notified when interesting things happen.
