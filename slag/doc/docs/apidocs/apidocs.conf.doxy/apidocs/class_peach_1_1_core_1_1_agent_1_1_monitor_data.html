Detailed Description

The information recorded by a monitor. This can be a fault or just an arbitrary collection of data.

The name of the agent that hosts the monitor that produced this data.

◆ Data

Dictionary<string, Stream> Peach.Core.Agent.MonitorData.Data
getset

A collection of arbitrary data recorded by the monitor.

Extra data to keep with teh fault. For each Key,Value pair, Value will be saved as "AgentName.MonitorName.DetectionSource.Key" Ownership of the stream is the responsibility of the monitor. The stream needs to be returned from GetMonitorData and remain open until the next call to IterationStarting or a call to SessionFinished.

◆ DetectionSource

string Peach.Core.Agent.MonitorData.DetectionSource
getset

The type of monitor that produced this data.

This field does not need to be set by the monitor. The agent server will set the detection source to the monitor's class name automatically. If a single monitor class has multiple internal detection sources, this field can be used to distinguish between them.
