Detailed Description

The State element defines a sequence of Actions to perform. Actions can cause a change to another State. Such changes can occur dynamically based on content received or sent by attaching python expressions to actions via the onStart/onComplete/when attributes.

void Peach.Core.Dom.State.MoveToNextAction ( )

Will move forward one action in execution.

This method can be called by state model mutators. If this method is not called and a different action is returned by the mutator, the current action scheduled for execution will be tried again on the next step through.
