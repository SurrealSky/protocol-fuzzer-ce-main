◆ ExpressionGet

string Peach.Core.Dom.Relation.ExpressionGet
getset

Expression that is run when getting the value.

This expression is only run when the data cracker has identified a size relation exists and is getting the value from the "from" side of the relation.

The expressionGet will only get executed when direcly calling the Relation.GetValue() method directly. It is not called from DataElement by design.

◆ ExpressionSet

string Peach.Core.Dom.Relation.ExpressionSet
getset

Expression that is run when setting the value.

This expression can be called numerouse times. It will be executed any time the attached data element re-generates it's value (internal or real).

The ExpressionSet is executed typically from DataElement.GenerateInteralValue() via Relation.CalculateFromValue(). As such this expression should limit the amount of time intensive tasks it performs.
