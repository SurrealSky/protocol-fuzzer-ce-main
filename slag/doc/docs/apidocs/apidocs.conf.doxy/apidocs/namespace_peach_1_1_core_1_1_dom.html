◆ ValueType

enum Peach.Core.Dom.ValueType
Enumerator
String 

Regular string. C style "\" escaping can be used such as: \r, \n, \t, and \\.

Hex 

Hex string. Allows specifying binary data.

Literal 

Treated as a python literal string. An example is "[1,2,3,4]" which would evaluate to a python list.

IPv4 

An IPv4 string address that is converted to an array of bytes. An example is "127.0.0.1" which would evaluate to the bytes: 0x7f, 0x00, 0x00, 0x01.

IPv6 

An IPv6 string address that is converted to an array of bytes.
