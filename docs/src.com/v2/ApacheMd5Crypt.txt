= crypto.ApacheMd5Crypt =

Apache style MD5 crypt. If no salt is provided, the first two chars of data are used. You can read more about link:Transformers.html[transformers here].

[source,xml]
----

<Block name="HelloWorld">
  <Transformer class="crypto.ApacheMd5Crypt"/>
  <Blob name="Data" value="Hello"/>
</Block>
----
