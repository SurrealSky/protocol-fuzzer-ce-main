== Tutorial: Dumb File Fuzzing with Fuzz Bang

Welcome to the dumb fuzzing tutorial, part of the Peach Quickstart.  In this tutorial we are going to fuzz mplayer using the Peach Fuzz Bang application.  This is a graphical application to get started fuzzing fast.  We will use a collection of PNG image files (seed files) with our fuzzing.
